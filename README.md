## 🧠 Project Overview

ElectraGuide is an intelligent web platform that helps users understand the election process in a simple, interactive, and accessible way. It combines AI, visualization, and personalization to guide users through every step of voting.

Whether you're a first-time voter or need clarity on procedures, ElectraGuide ensures you never feel lost.

## 🎯 Features

### 🤖 AI Assistant
- Conversational interface  
- Structured responses (steps, deadlines, documents)  
- Context-aware answers  

### 🗓️ Interactive Timeline
- Key election events  
- Visual flow of important dates  

### ✅ Step-by-Step Guide
- Checklist-based voting process  
- Progress tracking  

### 📍 Location-Based Insights
- Personalized election info  
- Region-specific timelines  

### 🔍 Smart FAQ
- AI-powered search  
- Preloaded election queries  

### ♿ Accessibility First
